8.5

A. Uma classe é o conjunto de objetos que possuem a mesma estrutura de dados, 
definida pelos atributos e métodos (comportamento), ou seja, é um modelo 
para criação de objetos com características em comum.

B. Um objeto é um elemento de uma classe, quando esta é instanciada.
Cada objeto tem sua identidade e se distingue de outros objetos a partir
de suas propriedades, mesmo que seus atributos e métodos sejam idênticos.

8.6 A assinatura de um método é o jeito de identificá-lo de forma única.
Métodos com mesmo nome (polimorfismo) podem se distinguir a partir da entrada de
parâmetros, por exemplo. Dentro de uma mesma classe, pode-se escrever dois métodos
de mesmo nome:

public void imprimir(){
	System.out.println("Oi");
}

public void imprimir(String frase){
	System.out.println(frase);
}

O método a ser executado vai depender do tipo de chamada (com ou sem parâmetros).
Se no código foi chamado o método "imprimir()", será impresso na tela: "Oi".
Mas se for colocado um String como input em "imprimir("Olá!")", será impresso na tela
a frase "Olá!".
